Blinkit Sales Analytics Dashboard

An interactive Power BI dashboard analyzing over $1.2M in sales data across outlet types, item categories, and geographic tiers.

Project Overview

This dashboard was built to help Blinkit (an instant delivery platform) understand sales performance across multiple dimensions — product categories, outlet sizes, and city tiers. It transforms raw sales data into clear, actionable business insights.

Features
Sales by Item Category — Identifies top-performing product types
Outlet Type Analysis — Compares performance across supermarkets and grocery stores
Geographic Tier Breakdown — Provides insights across Tier 1, Tier 2, and Tier 3 cities
KPI Tracking — Tracks Total Sales, Average Sales, and Item Ratings
Dynamic Filters — Interactive slicers for outlet size, location, and item type
DAX Measures — Custom calculations for deeper analysis
Tools & Technologies
Power BI Desktop — Dashboard creation and visualization
DAX — Custom measures and KPI calculations
Excel / CSV — Data source
Key Insights
Identified highest revenue-generating product categories
Revealed outlet size trends impacting sales volume
Highlighted Tier 3 cities as emerging high-growth markets
