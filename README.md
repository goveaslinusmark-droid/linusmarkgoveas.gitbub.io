🟡 Blinkit Sales Analytics Dashboard
An interactive Power BI dashboard analyzing over $1.2M in sales data across outlet types, item categories, and geographic tiers.

📊 Project Overview
This dashboard was built to help Blinkit (an instant delivery platform) understand their sales performance across different dimensions — product categories, outlet sizes, and city tiers. It transforms raw sales data into clear, actionable business insights.

🚀 Features

📦 Sales by Item Category — Identifies top-performing product types
🏪 Outlet Type Analysis — Compares performance across supermarkets and grocery stores
🗺️ Geographic Tier Breakdown — Insights across Tier 1, Tier 2, and Tier 3 cities
📈 KPI Tracking — Real-time metrics for Total Sales, Average Sales, and Item Ratings
🔍 Dynamic Filters — Slicers for outlet size, location, and item type
🧮 DAX Measures — Custom calculations for deeper business insights
